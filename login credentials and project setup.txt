To access this project

first download python 

then install xampp 

after installing xampp and setting up turn on Apache and mysql 

then go to phpmyAdmin via local host in your browser create a database named dbtheatre
then import the database in that table via the file in the folder DATABASE FILE 

then run these in the CMD 

pip install mysql-connector

pip install flask 

then access your project directory in the CMD
 
cd CinemaBookingSystem  

then run this command

python Cinema.py

then use the local host link in the cmd usually  http://127.0.0.1:5000

then paste it to the browser(Chrome, edge,....)
